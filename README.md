# 🎲 BingoMaster

Um projeto simples em **Windows Forms (C#)** para simular um sorteio de Bingo com 75 números.

## 📌 Funcionalidades

- Sorteio aleatório de números de 1 a 75.
- Exibição do número sorteado em destaque.
- Histórico dos números já sorteados.
- Reinício do jogo a qualquer momento.
- Interface gráfica amigável feita com Windows Forms.

## 🚀 Como executar

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/BingoMaster.git
   ```
2. Abra o projeto no **Visual Studio**.
3. Compile e execute (`F5`).

## 🖼️ Interface

- Um número grande no topo mostra o último sorteado.
- Botão **SORTEAR PRÓXIMO** para sortear o próximo número.
- Botão **REINICIAR JOGO** para reiniciar o globo de números.
- Lista com o **histórico dos sorteios**.

## 📂 Estrutura do projeto

- `Form1.cs` → Lógica principal do jogo.
- `Form1.Designer.cs` → Código gerado pelo designer da interface.
- `Program.cs` → Ponto de entrada da aplicação.

## 🔧 Tecnologias utilizadas

- **C#**
- **.NET Windows Forms**
- **Visual Studio**

## 📜 Licença

Este projeto é de uso livre para fins de estudo e aprendizado.
